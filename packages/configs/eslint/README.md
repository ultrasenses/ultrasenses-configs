# @ultrasenses/configs-eslint

Ultrasenses eslint config

## 🛠️ Installation

```
npm i @ultrasenses/configs-eslint --save-dev
```

```
yarn add @ultrasenses/configs-eslint --dev
```

## 💻 Usage

### in package.json

```json
"eslintConfig": {
  "extends": [
    "@ultrasenses/configs-eslint/main"
  ]
}
```
